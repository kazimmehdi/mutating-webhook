# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-10-24

### Added
- Initial release
- Mutating webhook server in Go
- Dynamic label configuration via ConfigMap
- Dynamic label configuration via environment variables
- Helm chart for easy deployment
- Automatic TLS certificate generation
- Health check endpoint
- Namespace filtering support
- Production-ready security settings
- Comprehensive documentation

### Features
- Add labels to pods dynamically
- Support for multiple configuration methods
- Zero-downtime label updates
- Kubernetes native deployment
